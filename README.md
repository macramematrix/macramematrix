# Macrame Matrix — macramematrix.com

Handcrafted sustainable macrame art by Monika A. Fernandes, Pune, India.

**Knotted with Precision**

## Tech Stack
- Plain HTML5, CSS3, vanilla JavaScript
- Hosted on Vercel with custom domain
- Optimized for performance, accessibility, and SEO

## Local Development
Open `index.html` in a browser, or use any local server:
```bash
npx serve .
```

## Deployment
Push to `main` branch → Vercel auto-deploys to macramematrix.com
